# Network Engineering Portfolio – James Paul Albornoz

This repository serves as a central index for my CCNA-level networking lab projects.  
Each project demonstrates a specific networking concept including VLAN segmentation, routing design, redundancy, and WAN failover.

These labs were built using **Cisco Packet Tracer** and documented to simulate real-world validation and troubleshooting workflows similar to those performed in NOC environments.
