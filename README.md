# Network Engineering Portfolio – James Paul Albornoz

This repository serves as the central index for my networking lab projects focused on **CCNA-level implementation, validation, troubleshooting, and operational workflows**.

The portfolio includes labs built in **Cisco Packet Tracer** and **GNS3**, with documentation structured to reflect practical validation, incident handling, and troubleshooting workflows relevant to **NOC** and **junior network operations** roles.
